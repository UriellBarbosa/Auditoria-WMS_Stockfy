# Roadmap

- Login + RBAC
- Registro de ocorrências (operador)
- Tratativa/relatórios (auditor/admin)
- Regras opcionais de endereço
- Export CSV

- Cadastro de itens (SKU)
- Movimentação entre endereços
- Ajuste de saldo
- Histórico de movimentações

## Fase 03 - Picking
- Lista de separação
- Baixa de itens
- Conferência

## Fase 04 - Inteligência
- Curva ABC
- KPIs e dashboards
- Divergências por setor/área

## Fase 05 - Integrações
- Importação CSV
- API para integração com WMS
- Webhooks (se suportado)
