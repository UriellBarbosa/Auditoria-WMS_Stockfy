# WMS Modular (Projeto Pessoal)

## Objetivo
 Construir um sistema WMS Modular (modelo stockfy), iniciando pelo módulo de Auditoria, com foco em uso real no ambiente de galpão e possibilidade de evolução para um produto comercial.

## Motivação
 Empresas com WMS ainda dependem de planilhas para auditoria e verificação de divergências de estoque. Este projeto cria um módulo de Auditoria que elimina a planilha e pode ser integrar a um WMS existente ou evoluir para um sistema WMS completo.

## Módulos planejados
- Módulo 01 (MVP): Auditoria
- Módulo 02: Estoqeu e Movimentação (entrada/saída, transferência, ajuste)
- Módulo 03: Picking e Separação
- Módulo 04: Relatórios e KPIs
- Módulo 05: Integrações (WMS terceiros / API)
